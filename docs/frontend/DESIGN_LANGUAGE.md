Engineering deserves elegance.

## Core Values

- Curiosity

## Design Principles

- Less, but better.
- Every interaction has a purpose.
- Typography is the primary visual element.
- Motion guides attention, never distracts.
- Whitespace is part of the design.
- Components are reusable and consistent.
- The website should feel discovered, not explained.
