# Portfolio CMS - Product Requirements Document (PRD)

## Overview

Portfolio CMS is a modern full-stack MERN application that allows developers to showcase their work through a beautiful public portfolio while managing all content from a secure admin dashboard.

The goal is to build a production-ready application that demonstrates full-stack development skills, clean architecture, authentication, CRUD operations, file uploads, API design, and deployment.

---

## Target Users

### Visitor
- View portfolio
- Browse projects
- Download resume
- Contact the developer

### Administrator
- Manage portfolio content
- Add/Edit/Delete projects
- Manage skills
- Update experience and education
- Upload resume
- View contact messages
- Manage profile settings
