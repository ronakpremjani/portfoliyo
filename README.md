# Portfolio CMS Client

A premium, high-performance portfolio built for software engineers, product builders, and creative professionals. It merges robust backend architectural principles with an exceptionally polished, interactive frontend experience.

## Overview
This project is engineered to feel intentional, reliable, and production-ready. It features a custom hybrid animation system using GSAP and Framer Motion, synchronized smoothly with Lenis.
